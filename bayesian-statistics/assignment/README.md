# 贝叶斯参数估计作业

本作业旨在评估你使用蒙特卡洛采样估计模型参数的能力。

你需要掌握：

- [ ] 创建并使用 [Scipy 的概率分布工具](https://docs.scipy.org/doc/scipy/reference/stats.html#continuous-distributions)
- [ ] [贝叶斯统计](../0_simple-bayesian-estimation.ipynb) 的基础知识
- [ ] 使用 [Kombine](../1_model-comparison-with-kombine.ipynb) 进行后验采样

完成作业后，你将学会如何利用不同实验收集的数据对多个模型进行参数估计，以及如何在包含异常值的数据中进行参数估计。
