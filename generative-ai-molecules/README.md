# 分子设计的生成式方法

生成式方法是机器学习技术，用于创建符合目标分布的数据。
例如，可以生成一组具有相似性质的分子，或生成一种策略，提出如何修改分子以改善其性质。
本模块将介绍一系列用于生成具有目标性质的新分子的方法。

## 学习目标

本模块的学习目标包括：

- 解释强化学习的理论与实践。*如何定义环境？什么是智能体？如何使用策略梯度（Policy Gradient）学习策略？*
- [...] 更多内容即将推出！

## 推荐论文

为了更好地理解该领域，建议阅读以下论文：

- [You 等，NeurIPS (2018)](https://arxiv.org/abs/1806.02473)：使用 PPO 和图卷积网络设计分子
- [Zhou 等，Sci Rep (2019)](https://www.nature.com/articles/s41598-019-47148-x)：使用 Q-learning 生成具有目标性质的分子
- Chris Yoon 关于策略梯度的[博客](https://medium.com/@thechrisyoon/deriving-policy-gradients-and-implementing-reinforce-f887949bd63)与[文章](https://towardsdatascience.com/understanding-actor-critic-methods-931b97b6df3f)：以简单图解说明策略梯度方法的原理
