# 详细安装说明

Linux 是这些讲义材料的首选操作系统，但（大多数）内容也可以在 Windows 或 OS X 上运行。  
各操作系统的特殊说明如下。

在 Windows 和 OS X 上，我们跳过安装 [`qml`](https://github.com/qmlcode/qml/)。QML 仅在[一个 notebook](../molecular-property-prediction/kernel-methods/fchl-in-one-notebook.ipynb) 中使用，且在非 Linux 系统上安装困难。
