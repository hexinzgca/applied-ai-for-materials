# 最优实验设计

最优实验设计方法可在实验进行过程中，通过推荐最有价值的实验方案，帮助您实时优化实验设计。  
本模块的笔记本将介绍相关基础知识和优化工具。

## 学习目标

本模块的学习目标包括：

- 解释采集函数（acquisition function）的目的与设计。*如何通过上置信界（Upper Confidence Bound）调节探索程度？*
- 针对不同问题选择最合适的算法类型。*高通量实验应采用哪种方法？*
- 开发用于测试主动学习系统的实验。*应使用什么作为基线？*
